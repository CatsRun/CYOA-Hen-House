# Choose Your Own Adventure: Hen House

In this CYOA game you are a hen. Each choice you are given allows you to explore your world. But, be careful, don't let the fox eat you!

## Instructions for Build and Use

[Software Demo](Put_Your_Video_Link_Here)

Steps to build and/or run the software:

1. Load the program in InteliJ
2. Select "run"
3. The program will run in the terminal

Instructions for using the software:

1. Select "run"
2. Use the terminal to interact with the game.
